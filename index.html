正在編輯：

新增合作公司（佔兩頁：左圖右文）

目前的公司頁面（依目錄順序）

會場與交通資訊（第二頁）

全書版面字體（即時預覽，套用到所有內文頁）

書本封面圖片

把 PNG / JPG / WebP 放進 index.html 旁的 assets/ 資料夾，填相對路徑（例：assets/cover.png）
進場動畫飛出的小書會自動同步成同一張封面 · 圖片載入失敗時自動退回內建封面

自訂影片進場動畫（三段式）

· 格式建議 WebM（VP9） 或 MP4，影片會被靜音自動播放（手機相容）
· 三段影片留空 = 使用內建的信封動畫（目前的效果）
· 無縫訣竅：A 的最後一格 = B 的第一格 = B 的最後一格 = C 的第一格
· 檔案放在 index.html 旁的 assets/ 資料夾，填相對路徑即可一起部署
· 儲存後重新整理頁面才會看到新的進場動畫

一鍵下載部署檔（最方便）

下載的檔案就是完整的新版網站，已包含你所有的編輯。
直接把它上傳到你的部署平台（覆蓋舊的 index.html）即可，不需要開程式碼。
· GitHub Pages：把檔案拖進 repo 覆蓋 → commit
· Netlify / Vercel：拖曳上傳即完成

備用方式：手動複製程式碼

若一鍵下載無法使用時的備案：複製程式碼 → 貼進 index.html 取代 資料標記之間的內容 → 上傳部署
